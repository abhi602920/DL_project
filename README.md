# 🚧 AI Industrial Safety Monitoring System

An AI-powered construction safety monitoring system that detects Personal Protective Equipment (PPE) compliance using YOLOv8 and provides intelligent safety assistance using Google's Gemini AI.

---

## 📌 Features

- 🦺 PPE Detection (Helmet & Safety Vest)
- 🤖 AI Safety Assistant (Gemini AI)
- 📷 Image Upload Detection
- 🎥 Webcam Detection
- 💬 AI Safety Chatbot
- 📊 Safety Status (SAFE / UNSAFE)
- 🧠 AI-generated Image Explanation

---

## 🛠️ Technologies Used

- Python
- YOLOv8 (Ultralytics)
- Streamlit
- Google Gemini AI
- OpenCV
- Pillow
- PyTorch

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/abhi602920/AI_Industrial_Safety_Project.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app/app.py
```

---

## 🔑 Environment Variables

Create a `.env` file.

```
GOOGLE_API_KEY=YOUR_GEMINI_API_KEY
```
